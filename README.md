# Sleep Apnea Project1

Offline Hybrid CNN-LSTM Model for OSA Event Detection Using Single-Lead ECG Signals

**Base Paper**: SE-MSCNN: A Lightweight Multi-scaled Fusion Network for Sleep Apnea Detection  
(Wang et al., IEEE Transactions on Biomedical Engineering, 2021)

## Project Goals
- Replicate baseline SE-MSCNN model (~89% accuracy)
- Add interpretability using SHAP
- Extend to multi-modal (ECG + SpO2)
- Improve fairness and efficiency
- Target conference submission (IEEE EMBC 2026 or similar)

## Dataset
Apnea-ECG Database (PhysioNet): https://physionet.org/content/apnea-ecg/1.0.0/
